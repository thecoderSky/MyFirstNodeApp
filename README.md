# MyFirstNodeApp
A simple Node.js application built using Express and deployed on Vercel. This project demonstrates backend setup, version control with GitHub, and cloud deployment.

#Features

* Basic Express server setup
* REST endpoint (`/`)
* Runs locally on port 3000
* Cloud deployment using Vercel
* GitHub integration

#Tech Stack

* Node.js
* Express.js
* Git & GitHub
* Vercel

# Project Structure

MyFirstNodeApp/
│── node_modules/
│── index.js
│── package.json
│── vercel.json
│── .gitignore
│── README.md

#Installation & Setup

#1. Clone the repository

git clone <your-repo-url>
cd MyFirstNodeApp

# 2. Install dependencies

npm install

# 3. Run the application

node index.js
